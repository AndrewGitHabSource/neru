Текст

Очень и очень длинный заголовок у типовой страницы, чтобы настроить междустрочный интервал. Данный заголовок нужен для того, чтобы настроить междустрочный интервал (H1)

Эта страница создана для демонстрации блоков и элементов, которые используются на сайте, и служит руководством для всех, кто работает над ним. Дизайнеры и технологи отрабатывают здесь стили, чтобы добиться приемлемых результатов в различных сочетаниях блоков и элементов. Контент-менеджеры и редакторы используют страницу в качестве справочника по верстке типовых страниц. Здесь же рассказывается о некоторых общих правилах оформления контента.

На сайте может встречаться несколько видов ссылок. Например, ссылка на Яндекс является внешней.
Внутренние ссылки могут открываться по умолчанию в текущем окне. Наиболее простыми способами выделения информации являются жирное и курсивное начертания.

А это стиль заголовка второго уровня. Несколько
двухстрочных заголовков (H2)

Далее следует заголовок третьего уровня. Для проверки
их сочетания (H3)

А именно: отступов и интерлиньяжа

Заголовок <h5> используется только для вспомогательной информации, где есть вероятность появления шестого уровня разбивки.
Пример — блок «Контактная информация», который внутри дробится по отделам компании.
Самый «нижний» заголовок <h6> предваряет такие элементы,
как «Комментарий» или «Таблица».

Списки

Для уменьшения расстояния между параграфом и связанным с ним списком, параграфу нужно добавлять класс «Перед списком» (before_list).

Нумерованный список:

  1. Первый пункт списка
  2. Второй пункт списка
  3. Третий пункт списка

Маркированный:

  • Первый пункт списка
  • Второй пункт списка
  • Третий пункт списка

Комментарий

Элемент «Комментарий» (класс comment) применяется к пояснительному тексту. И еще немного специально написанного текста,
чтобы получилось несколько строк для наглядности.

Важная информация

Элемент «Важно». (класс important), который используется для выделения особо важной информации. И еще немного специально написанного текста,чтобы получилось несколько строк для наглядности.
